# Лабораторная работа №10. Контейнеризация приложения

Папка содержит файлы, необходимые для запуска лабораторной работы №5 (Web API) в Docker-контейнере.

## Как запустить:

1. Откройте командную строку в папке `Lab10.Docker`.
2. Выполните команду (обратите внимание, что контекст сборки находится на уровень выше, это описано в yml файле):
   ```bash
   docker compose up -d --build
   ```
3. После успешного запуска проверьте работоспособность API по адресу:
   http://localhost:8080/swagger

**Примечание:** Так как в проекте используется ADO.NET без автоматических миграций (согласно измененным требованиям), таблицы в базе данных внутри контейнера будут отсутствовать при первом запуске. Для полноценной работы с данными необходимо выполнить скрипт инициализации таблиц к базе `LibraryDb` внутри поднятого SQL-контейнера.
